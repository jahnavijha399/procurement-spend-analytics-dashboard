# 📊 Procurement Spend Analytics Dashboard

## 🔹 Project Overview

This project is an end-to-end data analytics solution designed to analyze procurement spending and generate actionable business insights. It focuses on vendor performance, category-wise expenditure, payment tracking, and monthly trends to support better financial decision-making.

---

## 🔹 Objectives

* Analyze procurement spend across vendors and categories
* Identify high-cost vendors and departments
* Track monthly spending trends
* Monitor payment status (Paid vs Pending)
* Build interactive dashboards for business users

---

## 🔹 Tech Stack

* **Python (Pandas, NumPy)** – Data generation & cleaning
* **MySQL** – Data storage and SQL analysis
* **Power BI** – Interactive dashboard (3-page executive view)
* **Tableau** – Data visualization dashboard
* **Excel** – Data handling

---

## 🔹 Dataset

* Synthetic dataset of **10,000+ procurement records**
* Generated using Python (Faker library)
* Includes:

  * Invoice_ID
  * Vendor_Name
  * Category
  * Department
  * Invoice_Date
  * Amount
  * Payment_Status
  * Region
